Answers to Your Unasked Questions

What is Fool?

Fool is a software for editing live websites.

That so?

Yeah.

What is this, specifically?

This is a demo. If you type a web address in that bar up at the top, it'll tell you if Fool works with that website. If it does, you can try it out. Go ahead! Do it!

What does Fool cost?

For now, it doesn't cost anything! Additionally, plans to monetize Fool do not include licensing the software.

Why?

Because we at Fiddlenaut (the company that makes Fool) believe that the experience of editing websites sucks. So we're doing something about it.

Oh, did you mean why doesn't it cost anything? Because website editors shouldn't cost anything. The web is supposed to be accessible, and that means things like website editors should be easy to use and cheap.

Did you say this is Open Source? Can I has sauce?

You sure can! Here, check us out on Github. (Pun intended.)

Can I work on this with you? Like, for serious-like?

At this early stage, we don't have the capital to hire. But email us your resume (preferably in website format) if you're still interested. We'd be flattered!

Have you heard of PageCloud?

Yes, but does it matter? Fool: $0. PageCloud: more than that.

Hey, it broke!
